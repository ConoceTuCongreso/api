## Description
API for [ConoceTuCongreso](https://conocetucongreso.me).

## Instructions
Run `yarn install && yarn start` to run.

Run `yarn serve & yarn test` to run integration tests.

To start local Docker Postgres, run `./db-setup.sh`.
